Mis on Git?

Git on hajutatud versioonikontrollisüsteem, mis jälgib muudatusi mis tahes arvutifailide komplektis, mida tavaliselt kasutatakse programmeerijate töö koordineerimiseks, kes arendavad tarkvaraarenduse ajal koostöös lähtekoodi. Selle eesmärgid hõlmavad kiirust, andmete terviklikkust ja hajutatud, mittelineaarsete töövoogude toetamist (tuhanded paralleelsed harud, mis töötavad erinevates arvutites).
Git on kõige tuntuim ja kasutatuim versioonihaldusüsstem arendajate seas.

Võimalused

  • Jälgib ajalugu
  • Tasuta ja avatud lähtekoodiga
  • Toetab mittelineaarset arendust
  • Loob varukoopiaid
  • Skaleeritav
  • Toetab koostööd
  • Hargnemine on lihtsam
  • Hajutatud arendus
